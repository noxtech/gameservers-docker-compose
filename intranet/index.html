NoxLAN Game Servers

All servers are on the 192.168.100.x LAN network. Join from a PC connected to the same LAN.

🧠 How to Join Games

  • Unreal Tournament 99 / 2004, Insurgency, Counter-Strike (1.6 & Source): These should automatically appear in the in-game LAN browser.

Troubleshooting:

  1. Not seeing any LAN servers? This might be due to incorrect network adapter priority. If you have a VPN or virtual adapter, it may block discovery.
  2. Go to Control Panel → Network & Internet → Network Connections
  3. Right-click your LAN adapter → Properties → Internet Protocol Version 4 (TCP/IPv4) → Properties → Advanced
  4. Uncheck Automatic metric and enter 1 in the Interface metric box
  5. Alternatively, disable all other non-LAN adapters (VPN, virtual, etc.)
