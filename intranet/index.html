Troubleshooting:

  1. Not seeing any LAN servers? This might be due to incorrect network adapter priority. If you have a VPN or virtual adapter, it may block discovery.
  2. Go to Control Panel ‚Üí Network & Internet ‚Üí Network Connections
  3. Right-click your LAN adapter ‚Üí Properties ‚Üí Internet Protocol Version 4 (TCP/IPv4) ‚Üí Properties ‚Üí Advanced
  4. Uncheck Automatic metric and enter 1 in the Interface metric box
  5. Alternatively, disable all other non-LAN adapters (VPN, virtual, etc.)
